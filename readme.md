## 功能：

	1. 防撤回
	2. 去水印
	3. 多开 （工作需要，大家不要滥用。）

## 安装方式：

1. 先安装Xcode

2. cd 到项目目录，然后make 即可（推荐。）

   ```shell
   如果make失败，尝试输入
   sudo chown -R $(whoami) /Applications/企业微信.app
   然后再次make
   ```

3. 

4. 用Xcode打开WeWorkMacPlugin.xcodeproj，然后command+b编译后即可。

## 使用方式：

1. 防撤回、去水印 不需要任何操作，patch完程序就正常使用。
2. 多开  快捷键： ⇧⌘N 或者 Tabbar上的Plugin点击即可。
